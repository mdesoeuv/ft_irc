useful commands to test this project :

nc -c (ends line with /r/n) 127.0.0.1  (server address) 6697 (port)
nc -c 127.0.0.1 6697

suspend : CTRL+Z
resume : fg

see options in weechat :
/fset
/set

/set irc.server.ft_irc.nicks "john"
/set irc.server.ft_irc.username "john"
